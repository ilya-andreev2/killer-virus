# killer-virus

This repository contains the data and code for the K28/DUP240 project.

Ilya Andreev (ilya.andreev@nih.gov), Meru Sadhu (meru.sadhu@nih.gov)

## Description

T08 - Liquid growth assay on 16 diverse S. cerevisiae strains
T10 - BYxRM QTL mapping (segregant genotype data and QTL mapping algorithm was adopted from Gou et al. (Genetics, 2019))
T28 - YAR028W alleles
T30 - UIP3/YAR028W chimeragenesis

## Instructions

Download this repository and run the desired *T##-code.R* code in RStudio.
